# androidStudioTetris
## 環境

- Language : java
- Version : 8
- Android Studio :4.2.2

## アプリの概要
android環境で使用できるテトリスアプリです。
5個のボタンで左右移動と回転操作と高速落下とホールドの操作をそれぞれ行うことができます。
ルールは一般的なテトリスと同じで横一列にブロックが配置されたときに横一列ブロックが消去されスコアが加算されます。
